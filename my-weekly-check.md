> 파일 경로만 최신으로 바꾸면 매주 재사용 가능합니다.

---

## 1. 지난주 예산 집행 요약

```
@.claude/practice-data/D-경영/budget_2026.csv 와
@.claude/practice-data/D-경영/actual_Q2_2026.csv 를 비교해서
지난주 예산 집행 현황을 항목별 달성률 표로 요약해줘.
결론 먼저, 천원 단위, 이상 항목(110% 초과 또는 80% 미달)은 표시해줘.
```

---

## 2. 구매요청 우선순위 점검

```
@.claude/practice-data/D-경영/purchase_history.csv 에서
이번 주 납기 예정인 항목과 금액 상위 3건을 표로 정리해줘.
예산 잔여와 연결해서 우선 집행 순서를 제안해줘 (가설로 표기).
```

---

## 3. 설비투자 집행 현황 확인

```
@.claude/practice-data/D-경영/budget_2026.csv 에서
설비투자 항목의 연간 예산 대비 현재 달성률을 계산해줘.
하반기 집행 계획과 비교해서 리스크 여부를 표로 정리해줘.
```

---

## 4. 투자 심의 대기 항목 점검

```
지난 구매요청 중 아직 심의 완료되지 않은 항목이 있는지 확인해줘.
납기일 기준으로 긴급도를 상/중/하로 분류해서 표로 정리해줘.
```

---

## 5. 이상 항목 원인 가설 정리

```
이번 주 예산 대비 실적에서 이상 항목(110% 초과 또는 80% 미달)을
골라서 원인 가설을 각 2개씩 제안해줘.
확정안이 아닌 가설로 표기하고, 확인이 필요한 부서를 함께 적어줘.
```

---

## 6. 주간 경영 메모 초안 작성

```
위 1~5번 점검 결과를 종합해서
팀장 보고용 주간 경영 메모 초안을 작성해줘.
형식: 결론 요약(3줄) + 항목별 표 + 다음 주 확인 사항 3개.
내부 검토용, 백만원 단위.
```
